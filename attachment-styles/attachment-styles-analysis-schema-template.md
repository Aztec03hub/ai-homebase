# Attachment Styles Analysis Schema Template

## Document Structure

### Title Format
`[Partner 1 Name] & [Partner 2 Name] Attachment Styles Analysis - [Month Year]`

### Main Sections Structure

## 1. [Partner Name]'s [Attachment Style Type]

*Format: List primary attachment style with any secondary features (e.g., "XXX Attachment with YYY Features")*

### Example Entry Format

#### [Number]. [Descriptive Title] ([Date or Date Range])

**Initial Exchange:** *(or "Context and Complete Exchange:" for complex situations)*
- [Sender] ([YYYY-MM-DD HH:MM:SS AM/PM]): "[Complete message text]"
- [Note any time gaps like: [19 minute delay]]
- [Sender] ([YYYY-MM-DD HH:MM:SS AM/PM]): "[Complete message text]"

**Additional Context:** *(Include when needed for understanding)*
- [Relevant background information]
- [Related messages from other dates if they provide essential context]

**Resolution:** *(Include if the conflict/issue was resolved)*
- [Sender] ([YYYY-MM-DD HH:MM:SS AM/PM]): "[Resolution messages]"

**Analysis:** [Paragraph explaining how these messages demonstrate the attachment pattern. Should include:
- Specific attachment behaviors shown
- How this fits the attachment style
- Impact on the relationship dynamic
- Any patterns this exemplifies]

### Required Elements for Each Example

1. **Descriptive Title**: Clear, specific description of the attachment behavior
2. **Date/Time Context**: When this occurred in the relationship timeline
3. **Complete Messages**: 
   - Full sender name (not pronouns)
   - Exact timestamp
   - Complete message text (not truncated)
   - Note delays between messages when significant
4. **Sufficient Context**: Include enough messages before/after to understand the full situation
5. **Analysis**: Clear explanation linking the messages to attachment theory

## 4. Key Attachment Patterns Summary

### For Partner with Avoidant Attachment
Include numbered list of core patterns such as:
1. **Emotional Distance Strategies**
   - Delayed responses to emotional bids
   - Minimizing relationship milestones
   - Deflecting during vulnerable moments
   - Physical withdrawal from conflict

2. **Control and Independence**
   - Controlling how intimacy develops
   - Maintaining separate spaces/activities
   - Resistance to interdependence
   - Making unilateral relationship decisions

3. **Deactivation Strategies**
   - Physical/emotional withdrawal when overwhelmed
   - Changing subjects during emotional discussions
   - Creating conflicts to maintain distance
   - Compartmentalizing emotions

### For Partner with XXX Attachment
Include numbered list of core patterns such as:
1. **Pursuit and Reassurance Seeking**
   - Constant need for validation
   - Multiple messages when partner doesn't respond
   - Seeking public declarations of commitment
   - Repeated "I love you" declarations

2. **Hypervigilance and Jealousy**
   - Monitoring partner's activities and relationships
   - Catastrophizing about potential threats
   - Creating tests to verify loyalty
   - Extreme reactions to perceived competition

3. **Emotional Dysregulation**
   - Extreme reactions to perceived threats
   - Inability to self-soothe
   - All-or-nothing thinking
   - Self-attacking when feeling insecure

4. **Protest Behaviors**
   - Threatening behaviors when triggered
   - Excessive pursuit when partner withdraws
   - Unable to respect requests for space
   - Impulsive relationship decisions

### For Partners with Mixed Attachment Styles
If applicable, include patterns showing combination features, for example:
1. **Fearful-Avoidant/Disorganized Patterns**
   - Simultaneous approach and avoidance
   - Emotional dysregulation and dissociation
   - Hypervigilance combined with withdrawal
   - Push-pull dynamics
   - Trauma-informed responses

2. **Anxious Features in Avoidant Partners**
   - Fear of abandonment despite distancing
   - Jealousy expressed through control
   - Surveillance behaviors while maintaining distance
   - Catastrophizing while withdrawing

**Note:** Each pattern should reference specific examples from the analysis above

**Writing Guidelines for Pattern Summaries:**
- Use specific dates and incidents from the examples
- Quote key phrases that exemplify the pattern
- Show progression or consistency across the time period
- Connect patterns to attachment theory concepts
- Avoid introducing new examples not covered above
- Group related behaviors under clear categories
- Use consistent formatting for easy scanning

**Identifying Key Patterns:**
- Look for behaviors that repeat across multiple examples
- Note escalation or de-escalation of specific behaviors
- Identify trigger-response sequences
- Highlight self-awareness moments about patterns
- Include both problematic and growth-oriented patterns
- Consider cultural or family-of-origin influences

### Document Organization Order

1. **Title** - Clear identification of couple and time period
2. **Partner 1's Attachment Style** (with 6-8 detailed examples)
3. **Partner 2's Attachment Style** (with 6-8 detailed examples)
4. **Wonderful Connections & Healthy Growth** (with 8-10 positive examples)
5. **Key Attachment Patterns Summary** (consolidated patterns for each partner)
6. **Summary** (overall relationship assessment and trajectory)

**Transition Between Sections:**
- Each section should stand alone but reference patterns from other sections
- Use consistent terminology throughout
- Build complexity from individual examples to pattern summary
- End with forward-looking assessment based on all evidence presented

### Formatting Guidelines

#### Message Formatting
- **Bold** for section headers
- *Italics* for subsection descriptions or emphasis
- Timestamps in consistent format: (YYYY-MM-DD HH:MM:SS AM/PM)
- Em dashes for analysis points that need separation
- Block quotes only for external sources, not messages

#### Analysis Guidelines
- Use attachment theory terminology accurately
- Connect specific behaviors to attachment patterns
- Avoid judgmental language
- Acknowledge both partners' perspectives
- Note patterns across multiple examples
- Include both immediate and long-term impacts

**Effective Analysis Components:**
1. **Behavior Identification**: What specifically happened
2. **Attachment Link**: How this relates to attachment style
3. **Pattern Recognition**: How this fits broader patterns
4. **Impact Assessment**: Effect on partner and relationship
5. **Context Consideration**: Triggers or circumstances
6. **Growth Notation**: Any progress or regression shown

**Example Analysis Paragraph:**

When Phil expresses need for closeness ('Can we make it Facebook official?'), Alyssa's 19-minute delay and minimal response ('Sure ðŸ˜Š') shows discomfort with public intimacy markers. #### Context Requirements
- Include enough messages to show full interaction arc
- Note when messages are from different conversations
- Provide background for references to past events
- Explain relationship timeline when relevant
- Include resolution messages when available

### Handling Mixed Attachment Styles

When a partner shows significant features of multiple attachment styles:

1. **Primary vs. Secondary Features**
   - Identify the dominant pattern (most frequent/impactful)
   - Note secondary features that appear under specific triggers
   - Example: "Avoidant Attachment with Anxious Features"

2. **<Mixed attachment style name> Indicators**
   - Simultaneous contradictory behaviors
   - Rapid switching between attachment strategies
   - Trauma responses overriding attachment system
   - Inability to maintain consistent strategy

3. **Organization Approach**
   - Create separate sections for distinct patterns
   - Show how patterns interact or conflict
   - Note which triggers activate which pattern
   - Include examples of internal conflict

### Special Considerations

#### When Family is Involved
- Always include complete context about family members' roles
- Show how family dynamics trigger attachment responses
- Include any mentions of family patterns or history
- Note intergenerational transmission of attachment styles
- Include cultural factors that may influence attachment expression

**Example Family Pattern Documentation:**
- "Mother's doubts about partner triggered avoidant flight response"
- "Discovery that mother and grandmother met partners through same activity heightened anxious partner's fears"
- "Family's approach to conflict (silence/withdrawal) mirrors avoidant strategies"

#### For Complex Multi-Day Situations
- Use clear date headers to show progression
- Link related conversations across days
- Show how unresolved issues carry forward
- Include eventual resolutions even if days later

#### Sensitive Topics
- Include trigger warnings if needed
- Maintain respectful language about past trauma
- Show growth and healing when present
- Avoid graphic details unless essential for analysis

### Minimum Requirements per Person
- At least 6-8 examples of attachment style
- Mix of different types of situations
- Examples from across the time period
- Both minor and major incidents
- Resolution examples when available

### Key Attachment Theory Terms

**Avoidant Attachment:**
- Deactivation strategies
- Emotional distance/distancing
- Self-reliance
- Minimizing
- Compartmentalization
- Dismissive behaviors

**Anxious Attachment:**
- Hyperactivation strategies
- Protest behaviors
- Reassurance seeking
- Hypervigilance
- Emotional dysregulation
- Pursuit behaviors

**Fearful-Avoidant/Disorganized:**
- Approach-avoidance conflict
- Dissociation
- Emotional flooding
- Simultaneous conflicting needs
- Trauma responses
- Incoherent strategies

**Secure Behaviors:**
- Emotional regulation
- Effective communication
- Repair attempts
- Vulnerability with boundaries
- Consistent availability
- Earned security

### Quality Checks
- Each example can stand alone without reading others
- Sufficient context for someone unfamiliar with the couple
- Clear connection between messages and analysis
- Balanced representation of both partners
- Includes both challenges and growth
